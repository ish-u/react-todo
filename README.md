## Setup and Run Locally

- Clone this repository and install the Packages
  ```
  git clone https://github.com/ish-u/react-todo.git
  cd react-todo
  npm install
  ```
- To Run
  ```
  npm start
  ```
- Open [http://localhost:3000](http://localhost:3000) to view it in the browser.
